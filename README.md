# AI Study Buddy

AI Study Buddy is a Streamlit-based AI application that helps students:
- Explain topics
- Summarize notes

## Technologies Used
- Python
- Streamlit
- Transformers
- Torch

## How to Run
1. Activate virtual environment  
2. Install requirements  
3. Run Streamlit app
